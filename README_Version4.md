Quick start
1. Clone the repo:
   git clone https://github.com/G20-Invest/g20invest.git
   cd g20invest

2. Create the Vite + React app:
   npx create-vite@latest . --template react
   npm install
   npm install react-router-dom
   npm install --save-dev gh-pages

3. Start development:
   npm run dev
   # open http://localhost:5173

4. Build:
   npm run build
   npm run preview

5. Deploy to gh-pages:
   npm run deploy
